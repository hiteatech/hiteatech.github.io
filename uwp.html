Course Summary

UWP (Universal Windows Platform) is introduced in Windows 10 which is a common app platform available on every device that runs Windows 10.

The UWP provides a guaranteed core API layer across devices. This means you can create single app package that can be installed onto a wide range of devices.

Windows App Studio is a free online app creation tool that allows us to quickly to build Windows app to publish, test, and share. It allows us to create Windows app with no coding required.

In this event we will be building our own personalized news application.

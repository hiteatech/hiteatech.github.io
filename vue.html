Course Summary

Vue is a progressive framework for building user interfaces. Unlike other monolithic framework, Vue is designed from the ground up to be incremantally adoptable.

The core library is focused on the view layer only, and is very easy to pick up and integrate with other libraries as well.

In this event we will be building a simple to-do web app that will allow users to add and remove tasks.
  • Prerequisites and Requirements

    1. Online IDE - jsfiddle.net (No installation is needed)
    2. A little bit knowledge of JavaScript programming & JavaScript Framework

  • What Will I Learn?

    In the end of this course, you'll learn
    i. General understanding of Vue.js and Javascript framework in web development
    ii. How to implement Vue.js and its syntax
    iii. Build 2 simple web apps

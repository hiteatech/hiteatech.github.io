Course Summary

Learn the basics of Android and Java programming, and take the first step on your journey to becoming an Android developer!

This course is designed for students who are new to app development, and want to learn how to build Android apps. However, You need to have some Java programming experience.

Learning anything new can be tough. We will walk you through the process of making Android apps, but to get the most out of this course, you must bring your enthusiasm for learning, and budget time on your calendar to learn with us.

By the end of the course, you will build a simple app that you can share with your friends. We also hope that you will learn enough through this course to decide how best to continue your journey as an Android app developer, if you're interesting in pursuing such a path.

  • Prerequisites and Requirements

    1. Install Android studio & Java Dev kit JDK. Check Here
    2. Install Xamarin Android Player

  • What Will I Learn?

    List of learning outcome:

    1. Intro to android and java
    2. Android UI concepts
    3. Activity and Intents
    4. Fragments
    5. User Navigation
